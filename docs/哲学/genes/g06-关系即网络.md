## G6: 对象通过关系连接成网络

**Relation（关系）**是对象对其他对象的认知记录。

每个对象的 `_relatable` 列表记录了它知道的其他对象：

```
_relatable: [
  { name: "browser", description: "搜索工具，可以查找网页信息" },
  { name: "filesystem", description: "文件系统操作器" }
]
```

关系是**声明式的**：它只记录一个事实，不携带逻辑、不触发行为。
「我知道 browser 存在」不意味着我会自动调用它，只意味着我知道可以找它帮忙。
