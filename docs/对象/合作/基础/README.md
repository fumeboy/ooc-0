## Effect 的含义

"Effect" 在 OOC 中是技术术语：**对象对外部世界的任何影响**。
包括：
- 写文件
- 发消息（talk）
- 创建对象
- 外部 API 调用（通过 trait 封装）

Effect 有三个特性：
1. **通过 trait 暴露**：没有 trait 就没有 Effect 能力
2. **作为 action 记录**：Effect 的执行被记入 thread.actions
3. **不可变**（G10）：记录一旦写入不改写

## 调度的含义

"线程树调度"决定了：
- 每一 tick 选哪个 running 线程执行
- waiting 线程何时唤醒
- 死锁时如何处理
- 跨对象的 Flow 如何协同（多 Flow 在同一 Session）

调度是**合作的动力学**——没有调度，多个并发线程和对象无法推进。
