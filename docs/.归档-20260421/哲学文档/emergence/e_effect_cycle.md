# E11: Effect 循环——学习、创造、感知

**涉及基因**: G8(Effect) + G4(程序行动) + G7(持久化即存在) + G9(行为树)

三种 Effect 构成一个持续的循环：

```
我→我（自我演化） → 我→它（输出到世界） → 它→我（从世界学习） → 我→我 ...
```

具体而言：
1. **我→我**：对象修改自己的 traits/（添加新 trait、升级已有 trait）
2. **我→它**：对象带着改进后的能力进入任务，通过 sub-flow 与其他对象协作
3. **它→我**：其他对象的工作改变了 shared/ 中的内容，对象感知到新信息，沉淀为 trait

这个循环对应人类的：
- 学习/反思（我→我）
- 工作/创造（我→它）
- 经验/感知（它→我）

涌现效果：
- 对象的能力不是静态赋予的，而是在三种 Effect 的循环中螺旋上升的
- 每一轮循环，对象都是一个「新的自己」——有了更多经验、更成熟的 traits
- 系统整体的智慧是所有对象 Effect 循环的涌现结果
