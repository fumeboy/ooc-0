# todo/ — 待认领的迭代项

> 本目录**只存软链接**，指向 `../all/` 中的文件。

## 语义

- 出现在这里 = 待做（还未被认领）
- 消失 = 被认领（软链接移到 `doing/`）或被放弃（软链接被删除，all/ 仍保留）

## 不要在这里放物理文件

物理文件只在 `../all/` 下。这里的每个 `*.md` 应该是一个软链接：

```bash
ls -la todo/
lrwxr-xr-x  1 user  staff  26 ...  20260421_feature_X.md -> ../all/20260421_feature_X.md
```

## 查看待办列表

```bash
ls todo/
```

按创建时间（文件名前缀的 YYYYMMDD）排序。
